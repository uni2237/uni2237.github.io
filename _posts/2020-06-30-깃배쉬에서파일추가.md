바탕화면에 저장해둔 git파일에 파일 추가/변경  
혹은 깃배쉬에서 vi로

git add .  
git config user.name "이름"  
git config user.email "이메일"  
git commit -m "메세지"  
git push origin master  

+) 강제로 push하고 싶을때  
   git push -f origin master
